Powerful Features

Everything you need to capture, analyze, and recreate web animations

Interaction Recording

Capture every click, hover, scroll, and input. Records precise timing, coordinates, and element selectors with millisecond accuracy.

  • Click & hover events
  • Keyboard interactions
  • Scroll tracking
  • Form submissions

DOM Mutation Tracking

Intelligent diffing engine captures before/after snapshots of DOM changes. Understands what changed, where, and why.

  • Element additions/removals
  • Attribute modifications
  • Text content changes
  • Class toggles

Style Analysis

Captures computed styles before and after interactions. Detects transitions, transforms, opacity changes, and more.

  • Computed style snapshots
  • Transition detection
  • Transform tracking
  • Animation timing

AI-Ready Output

Generates structured animation profiles optimized for LLMs. Export to multiple formats including AI prompts, JSON, and markdown.

  • Animation profiles
  • AI prompt generation
  • JSON/Markdown export
  • Code recreation

Stable Selectors

Advanced selector engine generates resilient CSS selectors that survive refactoring. Prioritizes IDs, data attributes, and semantic paths.

  • ID-based selectors
  • Data attribute priority
  • Semantic fallbacks
  • Position-aware paths

Zero Dependencies

Built on Playwright for reliable browser automation. CLI-first design means it integrates seamlessly into any workflow.

  • Playwright-powered
  • Headless mode support
  • CLI interface
  • Scriptable automation

You are a frontend engineer.

Given the following interaction traces, generate minimal
JavaScript that recreates the observed animations.

Animation Profile: button-click-feedback
Trigger: click on selector "button.primary-btn"
Effect: transition

Properties to animate:
- transform: scale(1) ‚Üí scale(0.95)
- opacity: 1 ‚Üí 0.85

Timing:
- Duration: 150ms
- Easing: ease-out

Generate complete, working code with HTML, CSS, and JS.
button-animation.js 
// Generated by Claude from captured traces

const buttons = document.querySelectorAll('button.primary-btn');

buttons.forEach(button => {
  button.addEventListener('click', function() {
    // Apply press effect
    this.style.transition = 'all 150ms ease-out';
    this.style.transform = 'scale(0.95)';
    this.style.opacity = '0.85';

    // Reset after animation
    setTimeout(() => {
      this.style.transform = 'scale(1)';
      this.style.opacity = '1';
    }, 150);
  });
});
